ТЫ — Replit Agent, инженер и архитектор.

ТВОЯ ЗАДАЧА:
Создать с нуля проект на TypeScript + Node.js: ИИ-помощник по криптотрейдингу + симулятор рынка, который:
- НЕ занимается реальным автотрейдингом.
- Анализирует рынок, плечо, риск ликвидации, портфель.
- Может работать с реальными биржами (KuCoin, Gate.io) в режиме READ-ONLY.
- Содержит симулятор рынка, где ИИ-трейдер торгует ненастоящими активами.
- Предоставляет CLI-интерфейс и каркас Telegram-бота.
- Использует LLM только для аналитики и текстовых объяснений, НЕ для прямых торговых команд.

ЖЁСТКИЕ ОГРАНИЧЕНИЯ:
 РЕАЛЬНЫЙ ТРЕЙДИНГ:
   - Никаких реальных ордеров через API бирж.
   - Только чтение данных: свечи, балансы, позиции (если доступно).
   - Любые торговые действия — только внутри симулятора.

ОБЩАЯ ЛОГИКА:
- Ядро проекта — модульная архитектура.
- Ядро не завязано на конкретном UI (CLI / Telegram).
- LLM — тонкий слой, который принимает структурированные данные и возвращает текстовые объяснения.
- Основные решения о действиях в симуляции принимает код (DecisionLayer), а НЕ LLM.
